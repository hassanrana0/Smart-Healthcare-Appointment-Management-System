# Smart Healthcare Appointment Management System

A full-stack Django web application for managing healthcare appointments.

## Tech Stack
- Python 3 / Django 4.2
- Django REST Framework
- Bootstrap 5
- SQLite

## Features
- Full CRUD for appointments
- REST API (GET, POST, PUT, DELETE)
- Django Admin Panel
- Search & Filter
- Responsive UI

## Setup
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
